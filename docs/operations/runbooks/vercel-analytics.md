# Runbook: Vercel Analytics

## Overview
How to monitor application performance and traffic using Vercel's native analytics tools.

## Key Metrics to Monitor
- **Web Vitals:** LCP, FID, CLS scores for the landing page.
- **Serverless Function Executions:** Check for timeouts or high memory usage in App Router routes.
- **Traffic Spikes:** Identify source of unusual traffic during Game Jams.

## Access
Accessible via the Vercel Project Dashboard under the "Analytics" tab.
