# Planning Templates Reference

This directory contains templates, examples, and reference materials for the planning framework.

## Template Categories

### Executive Templates
- Executive summary template
- Stakeholder presentation template
- Project pitch template

### Planning Templates
- Scope definition checklist
- Architecture decision template
- Risk assessment matrix
- Resource allocation template
- Timeline template

### Reporting Templates
- Status report template
- Metrics dashboard template
- ROI calculation template
- Lessons learned template

## Usage

These templates are reference materials to help fill out the planning sections. They provide:
- Structure and format guidance
- Example content
- Best practices
- Common patterns

## Integration

Templates integrate with:
- Planning sections (0-13)
- Claude Skills (architecture-decisions, project-risk-identifier, etc.)
- Project documentation
- ADR (Architecture Decision Records)
