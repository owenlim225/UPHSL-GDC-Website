# Skills & Workflow Examples

Practical examples demonstrating how developers handle routine operational assignments locally within the GDC repository.

## Adding a New Dashboard Route

If an Officer requests a new page underneath their dashboard to manage "Events", you do not need to configure an Express router.
1. Create a server directory: `src/app/dashboard/admin/events/page.tsx`
2. Fetch data directly from Supabase server-side logic:
```typescript
import { createClient } from '@/utils/supabase/server';

export default async function EventsPage() {
    const supabase = await createClient();
    const { data: events } = await supabase.from('events').select('*');
    return <EventTable payload={events} />
}
```

## Adding Design System Tokens

If the club votes to migrate primary branding colors:
1. Navigate directly to `design-system.json`.
2. Modify the specific HEX values stored under the semantic key.
3. Because Tailwind utilizes CSS Variables mapped to this JSON output, no CSS files themselves require refactoring.

## Triggering Discord Webhooks

When writing background tasks that alert leadership:
1. Add an Edge Function (or server action):
```typescript
await fetch(process.env.DISCORD_WEBHOOK_URL, {
  method: 'POST',
  body: JSON.stringify({ content: "New member registered!" })
});
```
