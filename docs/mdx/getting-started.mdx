# Getting Started

Welcome to the **UPHSL Game Developers' Club (GDC)** platform source repository! This guide will help developers and club officers spin up the architecture locally in under 3 minutes.

## Quick Setup

### Step 1: Clone and Install

This project leverages a unified **Next.js** monorepo linked directly to a **Supabase** backend. 

```bash
git clone https://github.com/uphsl-gdc/website.git
cd website
npm install
```

### Step 2: Environment Variables

Request your local `.env.local` file from the Lead Technical Officer in the Discord Server, or configure your local edge variables:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role
```

### Step 3: Run the Development Server

Start the local Next.js App Router environment:

```bash
npm run dev
```

The GDC Public Website will be accessible at `http://localhost:3000`.
To view the Member Dashboard, visit `http://localhost:3000/dashboard`, which requires a test-user login.
